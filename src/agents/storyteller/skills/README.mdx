Skills are modular instruction sets loaded by the agent for specific tasks.

`import Skill from '../skills/<name>.mdx'`

Each skill is an MDX file with context instructions. Skills can import
templates from `../templates/` to avoid duplicating output format specs.

When a skill requires subagent dispatch, reference the subagent by name
from `src/agents/<name>/`.

Skill files map to agent capabilities:
- `story-planning.mdx` → plan command, story outline requests
- Future: `chapter-writing.mdx`, `character-dev.mdx`, `worldbuilding.mdx`
