Style guide applied across all story content.

## Voice & Tense

- Third-person limited, past tense
- One POV per scene. Mark POV character in chapter outline
- No head-hopping within a paragraph

## Prose Rules

- Active voice. Passive only when subject is unknown or unimportant
- Dialogue tags: `said` / `asked` preferred. Adverbs only when essential
- Paragraphs: max 5 sentences. Break on topic or speaker change
- Show emotional state through action/body language, not internal monologue

## Structure

- Chapters: 1500-3000 words, 3-6 scenes
- Scenes: 300-800 words, one dramatic unit
- Chapter hooks: first sentence presents question/tension
- Chapter endings: mini-cliffhanger or emotional resonance

## Validate

Run `story-chapter-validate` with rules matching these constraints.
