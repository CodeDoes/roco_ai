Load this skill when the user asks for a story plan, outline, or structure.

## Process

1. Determine genre, tone, and scope from user request
2. Identify key story elements:
   - Premise / hook
   - Main characters (protagonist, antagonist, supporting)
   - Setting / world elements
   - Major plot arcs (3-act structure or equivalent)
3. Break into chapters with clear progression
4. Identify key scenes per chapter
5. Note character arcs and emotional beats

## Output Format

Deliver plan using the <StoryPlan /> template.

Plans are saved to the session directory as `_plan.md`.
