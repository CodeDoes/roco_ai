Load this skill when summarizing existing story content or generating a synopsis
for a plan.

## Process

1. Read existing chapters from workspace
2. Extract: major plot events, character decisions, turning points
3. Condense each chapter to 2-3 sentences
4. Run `story-synopsis-analyze` for coverage gaps
5. Run `story-synopsis-validate` for completeness

## Guidelines

- Spoiler-free intro: premise + genre + tone
- Detailed section: chapter-by-chapter with key events
- Character arc summary: where each character starts and ends
- Keep synopsis proportional: 1 sentence per ~500 words of source
