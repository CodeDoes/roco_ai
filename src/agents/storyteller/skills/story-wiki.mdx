Load this skill when creating or updating worldbuilding entries.

## Wiki Entry Types

- **Character**: appearance, personality, backstory, relationships, arc
- **Location**: geography, significance, history, notable events
- **Faction**: goals, members, territory, conflicts
- **Magic/Technology**: rules, limitations, practitioners, cost
- **Timeline**: chronological events, age, durations

## Process

1. Determine entry type from context
2. Check existing wiki entries with `grep` to avoid duplication
3. Write entry following type template
4. Run `story-wiki-analyze` to verify structure
5. Run `story-wiki-validate` for completeness conditions

## Format

Each wiki entry is a markdown file in `workspace/<project>/wiki/<type>/<name>.md`.
