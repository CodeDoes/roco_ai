# Storyteller Agent

Prose generation agent for creative writing with RWKV.

## Capabilities
- Story generation (tell, continue, interactive commands)
- Chapter writing with named checkpoints
- Story planning (plan command)
- Paragraph break repair (--fix-paragraphs)

## Flow
```
cli.ts → StorytellerAgent
           ├── init() → loads instructions.md, bakes system prompt
           ├── continueStoryStream() → generate with streaming
           ├── saveChapterCheckpoint() → named state save
           └── loadChapterCheckpoint() → restore chapter context
```

## Session Data
Each session writes to `sessions/<ts>_<id>_<slug>/`:
- `session.jsonl` — JSONL event log
- `_system_baseline.state` — baked system prompt
- `_state_<name>.state` — named checkpoints
- `_plan.md` — story plan (generated by `plan` command)

## Tools
The storyteller exposes a read-only subset of tools (read, ls, grep, find)
and write for saving drafts.
